# Deep Learning–Based Breast Tumor Segmentation and Classification  from Ultrasound Imaging

## Overview
This work presents a segmentation-guided hybrid deep learning framework for breast ultrasound classification. A U-Net model is first used to localize tumor regions, and the extracted ROI images are then classified using multiple baseline models and a hybrid feature fusion model. The experimental results show that the proposed hybrid approach achieves high and balanced performance, with about 96% accuracy and 0.94 MCC. These results confirm that combining tumor segmentation with multi-model feature fusion can improve the reliability of breast cancer classification.
